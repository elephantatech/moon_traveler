Testing Checklist

Please test the following on your platform and report any issues:

  • Game launches and title screen displays
  • Model download menu shows installed + downloadable models
  • Model switching works and persists across restarts
  • talk command works — creature responds within ~10s
  • Creature responses are 2–4 sentences (not walls of text)
  • Sound effects play
  • Save / load works
  • dev mode shows game.log output
  • update command checks for updates
  • Ctrl+C exits cleanly

Report Issues
