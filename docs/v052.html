What's New

Six Features, Zero API Calls

<*>

ASCII Animations

A dedicated animation bar renders 2-line sprites in-place. Scan, travel, look, hazard, drone, and exchange — each with its own character. Toggleable via config animations off.

[O]

Evolving Drone

The travel drone sprite visually upgrades as you install components. Eyes go o→O, the belly fills with [] brackets — one pair per upgrade.

↑

In-Place Upgrades

Type update mid-game to check for new versions. Downloads the right binary for your platform, verifies integrity, and guides installation. Also: --upgrade flag.

█

Download Progress

Model downloads now show a TUI-compatible progress bar at 10% intervals. Ctrl+C during download offers to switch to a smaller model. Loading shows animated frames.

Δ

LLM Diagnostics

Dev mode now tracks every inference call: elapsed time, prompt/completion tokens, RSS memory delta. Last 5 calls visible in the panel. Logged to JSONL.

✓

Cross-Platform CI

Test matrix expanded to 3 OS × 2 Python versions = 6 jobs. Ubuntu, Windows, macOS. Python 3.11 + 3.12. Coverage reporting via pytest-cov.

Drone Sprites

Your Drone Evolves With You

Every upgrade you install changes the drone's appearance during travel. The sprite is rendered in the animation bar — a dedicated widget that updates in-place without scrolling the game log.

Changelog

Full Change List

  • new Dedicated #animation-bar widget for in-place 2-line sprite animations
  • new Drone travel sprite evolves with upgrades (eyes o→O, belly fills [])
  • new Binoculars sweep animation for look command
  • new Spinning sensor animation for scan command
  • new Hazard flash, exchange flash, drone transmit animations
  • new Drone conversation messages show in animation bar (private HUD)
  • new config animations on|off toggle + --disable-animation flag
  • new update command — check GitHub for new versions, download platform binary
  • new --upgrade CLI flag for headless update checks
  • new URL domain validation and path traversal protection in upgrade flow
  • new Download size verification for upgrade archives
  • new model command — re-download or switch AI model mid-game
  • new TUI-compatible model download progress (10% interval bars)
  • new Model loading animation in animation bar
  • new Ctrl+C during download offers to switch to a smaller model
  • new _timed_inference() wraps all 4 LLM call sites with timing
  • new log_llm_call() in dev mode — last 5 calls + JSONL logging
  • infra CI matrix: 3 OS × 2 Python = 6 test jobs
  • infra pytest-cov coverage reporting in CI
